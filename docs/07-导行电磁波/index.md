## 核心知识点

### 1. 导行波概论
- TEM波、TE波、TM波的概念
- 截止频率与截止波长
- 相速度和群速度

### 2. 矩形波导
- TE$_{mn}$模式与TM$_{mn}$模式
- 截止波数：$k_c = \sqrt{(m\pi/a)^2 + (n\pi/b)^2}$
- 主模TE$_{10}$的特性
- 单模传输条件

### 3. 圆波导
- 三种常用模式：TE$_{11}$、TM$_{01}$、TE$_{01}$
- 截止频率的确定（贝塞尔函数根）

### 4. 同轴线
- TEM模为主模
- 特性阻抗：$Z_0 = \frac{60}{\sqrt{\varepsilon_r}}\ln\frac{b}{a}$

### 5. 传输线理论
- 电报方程
- 特性阻抗与传播常数
- 反射系数与输入阻抗
- 史密斯圆图
- 阻抗匹配
