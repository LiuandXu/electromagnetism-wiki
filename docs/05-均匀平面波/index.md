## 核心知识点

### 1. 理想介质中的均匀平面波
- 波动方程的解：$\mathbf{E} = \mathbf{E}_0 e^{-j k z}$
- 波数：$k = \omega\sqrt{\mu\varepsilon}$
- 波速：$v_p = \frac{1}{\sqrt{\mu\varepsilon}}$
- 波阻抗：$\eta = \sqrt{\frac{\mu}{\varepsilon}}$（真空中 $\eta_0 \approx 377\Omega$）

### 2. 导电媒质中的均匀平面波
- 复传播常数：$\gamma = \alpha + j\beta$
- 衰减常数 $\alpha$ 与相位常数 $\beta$
- 趋肤深度：$\delta = \frac{1}{\alpha}$
- 良导体近似

### 3. 电磁波的极化
- **线极化**：$E_x$ 和 $E_y$ 同相或反相
- **圆极化**：$E_x$ 和 $E_y$ 幅度相等、相位差 90°
- **椭圆极化**：一般情况
- 极化判断方法：旋向（左旋/右旋）
