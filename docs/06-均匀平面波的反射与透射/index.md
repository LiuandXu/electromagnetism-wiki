## 核心知识点

### 1. 垂直入射
- 反射系数：$\Gamma = \frac{\eta_2 - \eta_1}{\eta_2 + \eta_1}$
- 透射系数：$\tau = \frac{2\eta_2}{\eta_2 + \eta_1}$
- 驻波比：$SWR = \frac{1+|\Gamma|}{1-|\Gamma|}$

### 2. 斜入射
- 斯涅尔反射定律：$\theta_r = \theta_i$
- 斯涅尔折射定律：$n_1\sin\theta_i = n_2\sin\theta_t$
- 全反射与临界角
- 布儒斯特角：$\theta_B = \arctan\sqrt{\varepsilon_2/\varepsilon_1}$

### 3. 平行极化与垂直极化
