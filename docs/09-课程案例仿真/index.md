# 第9章 课程案例仿真与实践


## 课程内容

### 9.1 课程案例概述

本章通过四个仿真案例，将电磁场与电磁波理论知识与工程实践相结合，加深对核心概念的理解与掌握。

**案例列表：**
1. 均匀平面波在有耗媒质中的传播
2. 电磁波的极化
3. 矩形波导的主模与基片集成波导
4. 对称振子天线的辐射特性仿真

### 9.2 案例1：均匀平面波在有耗媒质中的传播

**物理模型：** 无界有耗媒质（导电媒质）中均匀平面波的传播特性分析。

**关键参数：**
- 传播常数：$\gamma = \alpha + j\beta = j\omega\sqrt{\mu\varepsilon_c}$
- 复介电常数：$\varepsilon_c = \varepsilon - j\sigma/\omega$
- 衰减常数：$\alpha = \omega\sqrt{\frac{\mu\varepsilon}{2}\left[\sqrt{1 + (\sigma/\omega\varepsilon)^2} - 1\right]}$
- 相位常数：$\beta = \omega\sqrt{\frac{\mu\varepsilon}{2}\left[\sqrt{1 + (\sigma/\omega\varepsilon)^2} + 1\right]}$

**仿真要点：**
- 观察不同电导率 $\sigma$ 对衰减常数 $\alpha$ 和趋肤深度 $\delta$ 的影响
- 对比理想介质（$\sigma = 0$）与导电媒质中的波传播差异
- 验证良导体近似条件（$\sigma \gg \omega\varepsilon$）的适用范围
- 分析频率对趋肤深度的影响：频率越高，电磁波越难穿透导体

### 9.3 案例2：电磁波的极化

**物理模型：** 两个正交线极化波的合成，形成不同的极化状态。

**极化参数：**
- 轴比：$AR = E_{\max}/E_{\min}$（$1 \leq AR \leq \infty$）
- 倾角：$\tan 2\tau = \frac{2E_x E_y\cos\Delta\phi}{E_x^2 - E_y^2}$
- 旋向：由 $\Delta\phi$ 符号决定

**仿真要点：**
- 改变 $E_x/E_y$ 比值和相位差 $\Delta\phi$，观察极化轨迹变化
- $E_x = E_y, \Delta\phi = \pm 90^\circ$ 时获得圆极化
- 任意其他情况为椭圆极化
- 验证极化椭圆参数公式

### 9.4 案例3：矩形波导主模与基片集成波导

**矩形波导主模 TE$_{10}$ 仿真：**

- 截止频率：$f_c = 1/(2a\sqrt{\mu\varepsilon})$
- 场分布验证：$E_y$ 在宽壁中心最大，在窄壁为零
- 单模传输频率范围：$f_c < f < 2f_c$（对应 $a < \lambda < 2a$）
- 波导波长 $\lambda_g$ 与频率的关系

**基片集成波导（SIW）：**

基片集成波导是在介质基片上用金属化通孔阵列实现的类矩形波导结构，结合了微带线的小型化和矩形波导的低损耗优势，适用于毫米波集成电路。

### 9.5 案例4：对称振子天线辐射特性仿真

**仿真内容：**
- 不同长度对称振子的方向图（半波、全波等）
- 电流分布的可视化验证
- 输入阻抗随振子长度的变化
- 辐射电阻与振子长度的关系

**关键结论：**
- 半波振子（$2L = \lambda/2$）：$R_{rad} \approx 73\,\Omega$，方向图为 8 字形
- 振子长度增加，方向图主瓣变窄，但可能出现旁瓣
- 全波振子（$2L = \lambda$）：方向图在垂直于振子轴方向出现零点

**仿真工具建议：** HFSS、CST、FEKO 等电磁仿真软件可用于上述案例的三维全波仿真分析。
