## 实施步骤

### Step 1：源内容处理
- **操作**：源内容已在对话中完整提供，无需调用 `source_to_md.py` 转换
- **处理**：将用户提供的8页脚本作为 Markdown 源内容直接使用
- **验证点**：确认8页内容（封面 + 6页正文 + 总结页）结构完整

### Step 2：项目初始化
- **操作**：
  ```bash
  python ${SKILL_DIR}/scripts/project_manager.py init stealth_cloak_report --format ppt169
  ```
- **参数**：使用 ppt169 格式（1280×720），适合16:9宽屏投影
- **源内容导入**：由于内容是对话文本非文件，直接以对话上下文方式引用，不执行 `import-sources`
- **验证点**：项目目录结构在 `d:\学习\电磁场` 下创建成功

### Step 3：模板选项
- **决策**：默认自由设计路径。用户未提供模板目录路径，跳过模板流程
- **风格指导**：学术汇报风格，适合高校研究院所技术研讨场景

### Step 4：Strategist 阶段（⛔ BLOCKING — 需要用户确认）
- **操作**：
  1. 阅读 `references/strategist.md` 获取角色定义
  2. 阅读 `templates/design_spec_reference.md` 获取设计规格模板
  3. 通过 3 阶段 Confirm UI（`confirm_ui/server.py`）进行确认：
     - **Stage 1 — 方向锚定**：画布格式(ppt169)、受众(学术/技术)、视觉风格(推荐学术汇报风)
     - **Stage 2 — 设计系统**：页数(8页)、配色方案、字体方案、图标、公式渲染策略(推荐 mixed)
     - **Stage 3 — 图像/执行**：图像来源策略、AI图像风格、生成模式
  4. 公式处理：如果公式策略为 mixed/render-all，编写 `formula_manifest.json` 并运行 `latex_render.py`
  5. 输出 `design_spec.md` + `spec_lock.md`
- **注意**：Stage 2-3-4 是一次不间断运行，只在最终确认处 ⛔ BLOCKING 等待用户
- **关键考量**：
  - 内容含大量 LaTeX 公式（波阻抗、斯涅耳定律、相位梯度），需要 `latex_render.py` 渲染为 PNG
  - 学术风格建议：深蓝/科技蓝配色方案，中文字体可选思源黑体，公式用 mixed 策略
  - 此阶段为唯一用户确认关卡

### Step 5：图像获取阶段（条件性执行）
- **触发条件**：仅当 `design_spec.md §VIII` 中存在 `Acquire Via: ai` 或 `web` 的行
- **操作**：根据设计规格中的图像资源列表，按需执行 AI 生成或网络搜索获取
- **不触发条件**：如果已确认 `image_usage: none` 或仅 `formula`/`placeholder`，跳过此步

### Step 6：Executor 阶段（SVG 生成）
- **操作**：
  1. 阅读 executor 参考文件：`executor-base.md`、`shared-standards.md`、对应的 mode/visual-style 文件
  2. 启动 Live Preview（`svg_editor/server.py --live --daemon`）
  3. 预读布局模板 SVG 和图表模板 SVG
  4. **逐页生成** 8 页 SVG（`svg_output/`），每页生成前重读 `spec_lock.md`
  5. 运行 `svg_quality_checker.py` 质量检查
  6. 生成讲稿 `notes/total.md`
- **关键约束**：
  - SVG 必须手写（主 agent），不能使用脚本批量生成
  - 逐页生成，不能分组批处理
  - 每页生成前重读 `spec_lock.md` 获取颜色/字体/图标/图片配置
  - 公式已渲染为 PNG，在 SVG 中嵌入为图片引用

### Step 7：后处理与导出
- **操作**：
  1. `total_md_split.py` — 拆分讲稿到每页
  2. `finalize_svg.py` — SVG 后处理（图标嵌入、图片裁剪、文本扁平化等）
  3. `svg_to_pptx.py` — 导出为 PPTX 文件
- **输出**：`d:\学习\电磁场\exports\stealth_cloak_report_<timestamp>.pptx`
- **验证点**：PPTX 文件在用户工作区可访问
